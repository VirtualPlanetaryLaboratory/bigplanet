Command Line UsageÔÉÅ

Using bigplanet on the command line is relatively straight forward. After a suite of simulations (set up with vspace) has completed, simply run the following command in the in the command line:

bigplanet <input file> -c [number of cores] -m [email] -aoqv

where the input file is the bpl.in file, which is explained here.

There are three optional arguments:

-a : create a bigplanet archive

-c : set the number of processors to use

-m : emails the user when bigplanet is complete

-o : overwrite an existing archive

-q : quiet mode (nothing is printed to the terminal)

-v : verbose mode (all output is printed to the terminal)
